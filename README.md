# MetaCuenta – Guía práctica de Meta Business

Guía paso a paso, en español y orientada a usuarios no técnicos, que explica cómo crear una
cuenta de Meta y configurar Meta Business (crear negocio y portafolio comercial).

## 🌐 Ver la guía en línea

Una vez activado GitHub Pages (ver instrucciones abajo), la guía estará disponible en:

```
https://alesoander.github.io/MetaCuenta/
```

## 🚀 Activar GitHub Pages

Para publicar la guía como sitio web con GitHub Pages:

1. Ve a la página del repositorio en GitHub.
2. Haz clic en **Settings** (Configuración).
3. En el menú lateral izquierdo, selecciona **Pages**.
4. En la sección **Build and deployment**, configura:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main`
   - **Folder:** `/docs`
5. Haz clic en **Save**.
6. Espera unos segundos y recarga la página; GitHub te mostrará la URL del sitio publicado.

> **Nota:** Si la rama principal de tu repositorio se llama `master` en lugar de `main`,
> usa `master` en el paso 4.

## 📋 Contenido de la guía

La guía cubre el siguiente flujo:

1. Iniciar sesión en Meta Business (o crear cuenta nueva con Facebook).
2. Abrir Meta Business Suite.
3. Ir a **Configuración** (engranaje ⚙️) en el menú izquierdo inferior.
4. Crear un **Negocio** y completar información básica (sin datos fiscales ni verificación).
5. Crear un **Portafolio Comercial** desde el selector superior izquierdo.
6. Completar los datos del portafolio.
7. (Opcional) Agregar personas mediante invitación por email.

> En algunos casos el orden puede invertirse: primero el portafolio y luego el negocio.
